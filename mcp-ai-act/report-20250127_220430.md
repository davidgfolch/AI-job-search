## System Classification

### Why This System is High-Risk

The system falls under **Annex III, Section 4(b)** - "AI systems intended to be used for recruitment or selection of natural persons" as it:

1. **Automates CV Matching:** Uses AI agents (CrewAI) to match CVs against job requirements and calculate compatibility percentages
2. **Influences Employment Decisions:** The CV match percentage directly influences which jobs users consider applying for
3. **Processes Employment Data:** Handles personal CV data, job application history, and employment preferences
4. **Makes Recommendations:** AI-enriched data (salary estimates, skills extraction) guides user decisions

### System Components Analyzed:

1. **AI Enrichment Module** (`apps/aiEnrich/`)
   - Uses CrewAI agents with LLMs (Ollama/OpenAI)
   - Extracts salary estimates, required technologies, and skills from job postings
   - Processes job descriptions using natural language processing

2. **CV Matching Module** (`apps/aiEnrich/src/aiEnrich/cvMatcher.py`)
   - Compares user CVs against job requirements
   - Calculates match percentages (0-100) based on:
     - Required skills match (40% weight)
     - Experience level (25% weight)
     - Optional skills (15% weight)
     - Education/certifications (10% weight)
     - Industry knowledge (10% weight)
   - Stores match percentages in database

3. **Job Scraping & Management**
   - Aggregates job listings from multiple platforms
   - Tracks application status and user preferences
   - Manages employment-related personal data
