AI Job Search Scrapper

  • Specific site scrappers will load on respective tabs.
  • Be sure to pass security filters on each page in the first load.

Use the command line to run the scrapper with various options.

  • --wait: Wait for user input before starting the scrapper.
  • starting <ScrapperName>: Start the scrapper from a specific point.

Example usage:

python scrapper_main.py --wait starting Infojobs
